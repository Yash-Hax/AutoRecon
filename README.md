# Installation
Clone autorecon tool<br>
`git clone https://github.com/yash-hax/autorecon`<br><br>
Give grant permission to autorecon files<br>
`cd autorecon`<br>
`chmod +x *`<br><br>
Setup tool by installing required packages<br>
`./setup.py`<br><br>
Run autorecon tool<br>
`./autorecon.sh`<br><br>
